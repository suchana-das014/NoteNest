# NoteNest 📝

A simple Laravel-based note-taking application built while learning Laravel fundamentals and CRUD operations.

Users can:

* Register and log in
* Create notes
* Edit notes
* Delete notes
* Manage notes securely

---

## 🚀 Features

* User Authentication
* CRUD Functionality
* Responsive UI
* Laravel MVC Architecture
* Tailwind CSS Styling
* Alpine.js Interactivity
* SQLite Database Integration

---

## 🛠 Tech Stack

* Laravel 11

## ⚡ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/notenest.git
```

### 2. Move Into the Project Folder

```bash
cd notenest
```

### 3. Install Dependencies

```bash
composer install
npm install
```

### 4. Create Environment File

```bash
cp .env.example .env
```

### 5. Generate Application Key

```bash
php artisan key:generate
```

### 6. Run Database Migrations

```bash
php artisan migrate
```

### 7. Start Development Server

```bash
php artisan serve
npm run dev
```
